# NextImmo Marketing Website

Eigenständige Next.js-Marketingseite für `nextimmo.nexgen-consulting.de`.

## Lokal starten

```bash
npm install
npm run dev
```

## Vercel

1. Repository `MeisnerMax/NextImmo` als neues Vercel-Projekt importieren.
2. **Root Directory** auf `marketing` setzen.
3. Framework Preset `Next.js` verwenden.
4. Optional `NEXT_PUBLIC_SITE_URL=https://nextimmo.nexgen-consulting.de` setzen.
5. Nach dem ersten Deployment die Domain `nextimmo.nexgen-consulting.de` hinzufügen.

Die eigentliche Flutter-Anwendung im Repository bleibt dadurch unabhängig vom Marketing-Deployment.
