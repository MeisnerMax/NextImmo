## Notes

- Phase 2 standardizes the shared page structure first. Additional screens can be migrated onto the same templates without changing domain logic.
- Existing property-only screens such as `Market Comps` and `Operations Center` remain available, but now live inside the standardized detail template instead of isolated navigation.
